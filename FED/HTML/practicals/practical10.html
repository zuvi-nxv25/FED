Html elements are the building block of HTML page.with html constructs,image,and other objects such as interactive forms may be embedded into the rendered page.HTML provides a means to create structured documents by denoting structural semantics for text such as heading,paragraphs,lists,link,quotes and other iyems. HTML elements are delineated by tags, written using angle brackets Tags such as < img > and <input > directly introduce content into the page. Others tags such as <p > surround and provide information about document text and may include other tags as sub-element
Browsers do not display the HTML tags but use them to interpret the content of the page.
